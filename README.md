# TechFix IT Services Website

This project is a responsive website for an IT services company called **TechFix IT Services**.

## 🚀 Features

- Responsive design (desktop and mobile)
- Navigation menu with burger menu for mobile
- Services section
- Image carousel
- Contact section
- Modern UI with hover animations

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
